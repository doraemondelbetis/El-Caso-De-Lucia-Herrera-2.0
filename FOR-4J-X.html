FOR-4J-X — Módulo Forense

Sistema de Determinación de Culpabilidad | Clasificación Interna

Propósito Oficial
Este módulo FOR-4J-X tiene como finalidad analizar elementos del caso activo, correlacionar evidencia digital y testimonios, y generar un índice estimativo de culpabilidad para cada sospechoso registrado.
Parámetros de Análisis
• Coherencia de Testimonios • Presencia en Escena • Motivo Directo • Beneficio Potencial • Relaciones Interpersonales • Pruebas Materiales • Evidencia Digital
Índice Automático
El sistema asigna valores entre 0 y 100 por cada parámetro. Los resultados se integran en un Índice Final de Culpabilidad (IFC). Un IFC superior a 70 se considera ALTA PROBABILIDAD DE CULPABILIDAD.
Salida del Módulo
Salida: { Suspect: [Nombre], IFC: [0-100], Estado: [Descartado/Neutral/Probable/Alta] }
